# La Mision Tracker

Discover the technical intricacies that power La Mision Tracker, a platform dedicated to tracking and showcasing the La Mision trail race in Argentina.

## Technical Insights

La Mision Tracker is a product of precise engineering, employing cutting-edge technologies to ensure a seamless and captivating user experience. Let's dive into the technical details that bring this project to life:

### Technology Stack

- **Node.js:** The server-side backbone of La Mision Tracker, ensuring high-speed performance and responsiveness.

- **Vite:** The lightning-fast build tool for the frontend, optimizing the user interface and overall performance.

- **Bootstrap:** Seamlessly integrated for enhanced visual appeal, providing a polished and user-friendly experience.

- **React-Leaflet:** The dynamic mapping experience is powered by React-Leaflet, allowing users to navigate and explore race routes effortlessly.

- **MySQL Database:** The cornerstone of data management, MySQL efficiently handles crucial information about runners, routes, and race details, ensuring robust and secure data storage.

### User Interaction

- **Interactive Maps:** Powered by React-Leaflet, our maps provide users with a dynamic and engaging experience, allowing them to explore La Mision race routes with ease.

- **Elevation Profiles:** Gain insights into the race terrain with detailed elevation profiles, assisting runners in preparing for the challenges ahead.

- **Real-Time Runner Information:** La Mision Tracker dynamically fetches and displays real-time information about each runner, fostering a deeper connection between participants and spectators.
