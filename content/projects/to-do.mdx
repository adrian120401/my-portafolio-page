# To-Do App: Overview

The To-Do App is a task management application designed to streamline your daily tasks. With features like task creation, editing, and deletion, as well as customizable lists, this app brings efficiency to your task organization. Built using React, Java, and Firebase, it ensures a robust and scalable experience.

## Features

- **Create, Edit, and Delete Tasks:** Effortlessly manage your tasks with the ability to create, edit, and delete them as needed.

- **Organize Tasks into Customizable Lists:** Tailor your task organization by creating customizable lists, allowing you to categorize and manage your tasks effectively.

- **User Authentication via Google Sign-In:** Securely log in using Google sign-in through Firebase, ensuring a seamless and trustworthy user experience.

- **Data Storage and Retrieval with Cloud Firestore:** Benefit from efficient data storage and retrieval through Cloud Firestore, providing a reliable database for your task and list information.

## Technologies Used

The To-Do App is crafted with a powerful tech stack to deliver a smooth and feature-rich experience:

- **React:** The front-end of the application is built using React, a popular JavaScript library known for its efficiency in creating user interfaces.

- **Java:** The back-end of the application is developed in Java, utilizing Spring Boot, a framework known for building robust and scalable web applications.

- **Firebase:** Leveraging Firebase for user authentication, and Cloud Firestore for data storage, the To-Do App ensures a secure and efficient environment.
