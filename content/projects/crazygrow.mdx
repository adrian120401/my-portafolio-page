# CrazyGrow E-commerce

A full-featured e-commerce platform developed for CrazyGrow, a leading grow shop in Uruguay. This project delivers a comprehensive online shopping experience with modern technologies and robust functionality.

## Key Features

### E-commerce Essentials

-   Product catalog with advanced filtering and search capabilities
-   Shopping cart with real-time updates
-   Secure checkout process
-   User account management
-   Order tracking system
-   Inventory management

### Admin Dashboard

-   Product management interface
-   Order processing and tracking
-   Inventory control
-   Sales analytics and reporting
-   Customer database management

### Technical Features

-   Responsive design for all devices
-   SEO optimization
-   Fast loading times
-   Secure payment gateway integration
-   Real-time inventory updates

## Technology Stack

-   **Frontend:**
    -   React for a dynamic user interface
    -   Tailwind CSS for modern and responsive styling
    -   Context API for state management
-   **Backend:**
    -   Nest.js for robust API development
    -   PostgreSQL for reliable data storage
    -   JWT for secure authentication
-   **Infrastructure:**
    -   Azure for hosting and deployment
    -   GitHub Actions for CI/CD pipeline
    -   Automated testing and deployment workflows

## Business Impact

The platform has transformed CrazyGrow's business operations by:

-   Expanding their market reach beyond physical locations
-   Streamlining order processing and inventory management
-   Providing valuable insights through analytics
-   Improving customer satisfaction with a seamless shopping experience
