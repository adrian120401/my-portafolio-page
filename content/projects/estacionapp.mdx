# EstacionApp

Discover EstacionApp, your ideal companion for solving parking challenges in urban environments. This web application simplifies finding and reserving parking spaces, offering the convenience of locating and booking available spots. Whether you need instant parking or want to monetize your unused space, EstacionApp makes it easy. Explore the freedom of parking with us!

## Technical Overview

EstacionApp is built with a robust stack of modern technologies to ensure a seamless and efficient user experience. Here's a detailed look at the technologies powering EstacionApp:

### Technology Stack

- **Frontend:**
  - **React:** For building dynamic and responsive user interfaces.
  - **Vite:** Ensures fast development and optimized build processes.
  - **TailwindCSS:** Provides a sleek and modern design with utility-first CSS.

- **Backend:**
  - **Node.js:** The runtime environment for executing server-side code.
  - **Express:** A minimalist web framework for Node.js, facilitating robust and scalable backend development.
  - **Sequelize:** An ORM for Node.js, making database interactions with MySQL smooth and efficient.
  - **MySQL:** A reliable database system for managing parking space data and user information.

- **UX/UI Design:**
  - **Figma:** Used for designing intuitive and user-friendly interfaces.

## Key Features

- **Search and Reserve:** Quickly find and book available parking spaces with ease.
- **Monetize Your Space:** Rent out your unused parking spots and earn extra income.
- **User-Friendly Interface:** Navigate through the application effortlessly with a clean and intuitive design.
- **Real-Time Availability:** Stay updated with real-time information on parking space availability.
